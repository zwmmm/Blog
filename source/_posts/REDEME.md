### step
1. `npm install -g hexo-cli`
2. `npm install hexo-deployer-git --save`

### 新建文章
`hexo new <title>`
1. banner 添加帖子图
1. thumbnail 添加缩略图

### 发布流程
`hexo clean && hexo g && hexo d`

### 本地预览
`hexo s`

### 修改主题
1. 找到主题的github地址
2. git clone url themes/name
